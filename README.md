## About

A complete, responsive marketing site with all the pages a product company
needs: home, about, features, careers (with a role-detail page), blog listing,
contact, FAQs, help center, downloads and a custom 404.

## Tech stack

- **HTML5** — semantic, multi-page structure
- **CSS3** — custom styles plus `animate.css` for transitions
- **Bootstrap** — responsive grid and components

## Running locally

This is a fully static site — serve the folder with any static server:

```sh
npx serve .
```

Then open the printed local URL.
